The American Red Cross and the British Red Cross are working together because they both love making maps and helping people, yay! This page shows a lot of the maps we've made for folks in the field and our operations centers.

Please report any issues to Daniel Joseph or using GitHub.

How to use the mapfolio: Double click the map to zoom in on an area of interest. Click a marker on the map or a thumbnail in the gallery below to preview the map and get the option of downloading a PDF. Use the filter buttons to explore this collection. The extent filters will return maps from any of the checked areas. For example you can search for maps that are from either "Ormoc" or "Tacloban." The type filters will return only maps tagged with every checked type. For example you can search for maps that are tagged as both "OSM" and "Poster."
