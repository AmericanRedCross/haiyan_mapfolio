The American Red Cross and the British Red Cross are working together because they both love making maps and helping people, yay! This page shows a lot of the maps we've made for folks in the field and our operations centers.

Filters: Use the buttons to explore this collection. The extent filters will return maps from any of the checked areas. For example you can search for maps that are from either "Ormoc" or "Tacloban." The type filters will return only maps tagged with every checked type. For example you can search for maps that are tagged as both "OSM" and "Poster."
